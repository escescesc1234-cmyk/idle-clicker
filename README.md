# 구름 멍때리기

아무 생각 없이 구름을 모으는 아이들 클리커 + 멍 때리기 공간.

## 플레이

GitHub Pages: 배포 후 이 README 상단에 URL이 표시됩니다.

## 로컬 실행

```powershell
cd idle-clicker
python -m http.server 8765 --bind 0.0.0.0
```

브라우저에서 `http://localhost:8765` 접속.

## 같이 멍하기

1. **방 만들기** → 방 코드 또는 링크 공유
2. 친구가 같은 사이트 주소에서 **참가**
3. 방장은 창을 켜 둔 채로 기다리기
